# DocAppoint Client Side

Frontend for the DocAppoint doctor appointment application built with Next.js.

## Features

- Authentication system
- Doctor appointment booking
- Protected routes
- Dashboard
- Responsive design
- Dark & light theme
- Animated UI

## Technologies

- Next.js
- React.js
- Tailwind CSS
- DaisyUI
- HeroUI
- Framer Motion
- Better Auth
- MongoDB
